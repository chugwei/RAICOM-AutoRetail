# 睿抗机器人大赛 - 自主移动抓取机器人系统

## 简历项目介绍

```
2024.03-2024.08              睡抗机器人大赛-自主移动抓取机器人系统                      核心算法开发
技术栈：Python, C++, ROS, OpenCV, YOLOv5, ONNX, PyMyCobot, GMapping SLAM, move_base, 超声波

视觉检测与伺服抓取：基于YOLOv5+ONNX部署轻量化检测模型，实现苹果/时钟等物体实时识别（置信度>0.85）；设计宽度-距离映射算法，结合超声波闭环反馈实现厘米级抓取定位。

SLAM建图与自主导航：基于GMapping算法实现激光SLAM建图，构建栅格地图；集成move_base导航栈实现全局/局部路径规划与自主避障，支持手柄人工接管切换。

多模块协同调度：设计"建图→导航→视觉搜索→抓取→分类放置"主循环框架，基于Flask开发远程任务服务器，实现全流程自动化。
```

---

## 技术方案要点

| 模块 | 技术 | 说明 |
|------|------|------|
| 目标检测 | YOLOv5s ONNX | 5类物体，320×320输入 |
| SLAM建图 | GMapping | 激光雷达栅格地图构建 |
| 自主导航 | move_base | 全局/局部路径规划 |
| 定位 | 超声波HC-SR04 | 距离误差<2cm |
| 机械臂 | PyMyCobot | 6轴控制，夹爪操作 |
| 远程控制 | Flask + ngrok | 任务下发与监控 |

## 核心流程

```
SLAM建图 → 导航定位 → YOLO视觉搜索 → 执行抓取 → 转身返回 → 分类放置 → 循环
```
